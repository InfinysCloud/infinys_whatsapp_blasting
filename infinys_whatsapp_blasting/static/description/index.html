Whatsapp Blasting / Broadcasting Messaging

About the Module

This module provides a Whatsapp Blasting / Broadcasting Messaging module is a powerful communication tool designed to send personalized or bulk messages to multiple WhatsApp contacts directly through a user-friendly dashboard.

Ideal for businesses, marketers, and customer support teams, this tool leverages the WhatsApp Web protocol to automate message delivery while maintaining human-like interactions.

Whatsapp Blasting Distribution is a system designed to automate and streamline the process of sending bulk messages to a large list of WhatsApp contacts. It is commonly used for marketing campaigns, announcements, customer engagement, alerts, and internal communications. free, easily accessible, and secure with an API key, so it can be used by businesses, marketers, and customer support teams for various application needs.

All In One WhatsApp Features :

  • Configuration : Use WhatsApp Web API / WhatsApp Business API
  • Bulk Messaging: Send messages to multiple contacts at once.
  • Scheduling: Plan and schedule messages for optimal delivery times.
  • Effectively reach customers through personalized messages
  • Record all incoming and outgoing messages for compliance and analysis.
  • Automatically convert all incoming messages and save it as database contact
  • Respond to user messages through the app, no need jump to another whatsapp client
  • Analytics: Track message delivery and engagement metrics with report (not this in community edition).

Mailing List

A Mailing List is a group of contacts (subscribers) that you can send email campaigns to. It works like a "bucket" where you store and organize your audience before launching an email blast. Example mailing list like VIP Customers, Webinar Attendees, Newsletter Subscribers, etc.

In this menu, you can create, edit, and delete mailing list and also have address book of contacts, log work incoming / outgoing whatsapp. It will make us more easily to organize it and see all the whatsapp log transaction

For incoming whatsapp, you can also see all detail of message and respond / reply the message as well.

Mailings

Mailing in Odoo is the feature used to design, send, and track whatsapp campaigns directly from your Odoo system.
It allows businesses to communicate with customers, prospects, or subscribers in a professional and automated way. You can create, edit, and delete mailings, and schedule them to be sent at a specific time.
